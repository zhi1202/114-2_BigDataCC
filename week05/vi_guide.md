# 第 5 週補充教學：使用 vi 文字編輯器

## 為什麼學 vi？

nano 簡單易學，但 vi 才是 Linux 世界中最普遍的編輯器。幾乎所有 Linux 系統都預裝 vi，即使在最精簡的伺服器環境中也一定找得到。學會 vi 讓你在任何 Linux 機器上都能編輯檔案。

| 比較 | nano | vi |
|------|------|-----|
| 學習難度 | 簡單，直覺操作 | 需要熟悉模式切換 |
| 系統預裝 | 大部分有，少數沒有 | 幾乎所有 Linux 都有 |
| 操作效率 | 一般 | 熟練後非常快速 |
| 業界使用率 | 入門用 | 非常廣泛 |

## 第一部分：vi 的核心觀念「模式」

vi 跟 nano 最大的不同是它有**模式**的概念。你必須先理解這件事，否則會覺得 vi 很奇怪。

### 三種模式

```
              按 i / a / o
一般模式 ──────────────────► 插入模式
（Normal Mode）              （Insert Mode）
   ◄──────────────────
              按 Esc

   │
   │ 按 :
   ▼

命令模式
（Command Mode）
   按 Enter 執行後自動回到一般模式
```

**一般模式**：開啟 vi 時的預設模式。在這個模式下，鍵盤按鍵是指令而不是輸入文字。例如按 `dd` 是刪除整行，不是輸入 dd。

**插入模式**：跟 nano 一樣，按鍵盤就是輸入文字。畫面左下角會顯示 `-- INSERT --`。

**命令模式**：輸入冒號 `:` 進入，用來儲存、離開、搜尋等操作。

### 最重要的一件事

**不知道自己在哪個模式時，按 Esc 就對了。** Esc 永遠會把你帶回一般模式。

## 第二部分：vi 基本操作

### 2.1 開啟檔案

```bash
# 建立或開啟檔案
vi myfile.txt

# 開啟已存在的檔案
vi ~/intro.txt
```

### 2.2 進入插入模式（開始打字）

開啟 vi 後，你在一般模式，不能直接打字。按以下任一鍵進入插入模式：

| 按鍵 | 功能 |
|------|------|
| `i` | 在游標前面開始輸入 |
| `a` | 在游標後面開始輸入 |
| `o` | 在游標下方新增一行並開始輸入 |
| `O` | 在游標上方新增一行並開始輸入 |

進入插入模式後，左下角會顯示 `-- INSERT --`，這時就可以正常打字了。

### 2.3 回到一般模式

按 `Esc` 鍵。左下角的 `-- INSERT --` 會消失。

### 2.4 儲存與離開

在一般模式下，按 `:` 進入命令模式，然後輸入以下指令：

| 指令 | 功能 | 說明 |
|------|------|------|
| `:w` | 儲存 | Write，儲存檔案 |
| `:q` | 離開 | Quit，離開 vi |
| `:wq` | 儲存並離開 | 最常用的組合 |
| `:q!` | 強制離開不儲存 | 改壞了想放棄修改時使用 |
| `:wq!` | 強制儲存並離開 | 覆寫唯讀檔案時使用 |

輸入指令後按 `Enter` 執行。

### 2.5 完整操作流程

```
vi myfile.txt        ← 開啟檔案（進入一般模式）
      │
      │ 按 i
      ▼
開始打字...           ← 插入模式，正常輸入文字
      │
      │ 按 Esc
      ▼
回到一般模式          ← 準備儲存或執行其他指令
      │
      │ 輸入 :wq
      ▼
儲存並離開            ← 回到終端機
```

## 第三部分：vi 常用指令

### 一般模式下的指令

#### 游標移動

| 按鍵 | 功能 |
|------|------|
| `h` | 左移 |
| `j` | 下移 |
| `k` | 上移 |
| `l` | 右移 |
| 方向鍵 | 也可以用方向鍵移動 |
| `gg` | 跳到檔案第一行 |
| `G` | 跳到檔案最後一行 |
| `0` | 跳到行首 |
| `$` | 跳到行尾 |

#### 刪除

| 按鍵 | 功能 |
|------|------|
| `x` | 刪除游標所在的字元 |
| `dd` | 刪除整行 |
| `3dd` | 刪除 3 行 |
| `dw` | 刪除一個單字 |

#### 複製與貼上

| 按鍵 | 功能 |
|------|------|
| `yy` | 複製整行 |
| `3yy` | 複製 3 行 |
| `p` | 在游標下方貼上 |
| `P` | 在游標上方貼上 |

#### 復原與重做

| 按鍵 | 功能 |
|------|------|
| `u` | 復原上一步 |
| `Ctrl+r` | 重做 |

### 命令模式下的指令

| 指令 | 功能 |
|------|------|
| `:w` | 儲存 |
| `:q` | 離開 |
| `:wq` | 儲存並離開 |
| `:q!` | 不儲存強制離開 |
| `:set number` | 顯示行號 |
| `:set nonumber` | 隱藏行號 |
| `/關鍵字` | 搜尋文字，按 `n` 跳到下一個 |

## 第四部分：用 vi 完成 Week05 作業

### 作業第 1 題：用 vi 建立自我介紹

nano 的做法：
```bash
nano ~/intro.txt
```

vi 的做法：
```bash
vi ~/intro.txt
```

操作步驟：

```
1. 輸入 vi ~/intro.txt            ← 開啟檔案
2. 按 i                           ← 進入插入模式
3. 輸入以下內容：
   姓名：XXX
   學號：XXX
   系所：XXX
   興趣：XXX
   修這門課的期望：XXX
4. 按 Esc                         ← 回到一般模式
5. 輸入 :wq 按 Enter              ← 儲存離開
6. 執行 cat ~/intro.txt            ← 確認內容
```

新增日期那行：

```
1. 輸入 vi ~/intro.txt            ← 開啟檔案
2. 按 G                           ← 跳到最後一行
3. 按 o                           ← 在下方新增一行並進入插入模式
4. 輸入：今天日期：2026-03-24
5. 按 Esc                         ← 回到一般模式
6. 輸入 :wq 按 Enter              ← 儲存離開
7. 執行 cat ~/intro.txt            ← 確認內容
```

作答時第 3 個任務要列出快捷鍵，vi 的版本：

```
=== 任務 3：說明你使用了哪些 vi 快捷鍵 ===
1. i 進入插入模式
2. Esc 回到一般模式
3. :wq 儲存並離開
4. G 跳到最後一行
5. o 在下方新增一行
6. dd 刪除整行
```

### 作業第 2 題：用 vi 建立 Shell Script

nano 的做法：
```bash
nano ~/my_script.sh
```

vi 的做法：
```bash
vi ~/my_script.sh
```

操作步驟：

```
1. 輸入 vi ~/my_script.sh         ← 開啟檔案
2. 按 i                           ← 進入插入模式
3. 輸入 Script 內容：
   #!/bin/bash
   echo "========================================="
   echo "  系統資訊報告"
   echo "========================================="
   echo "使用者：$(whoami)"
   echo "主機名稱：$(hostname)"
   echo "目前日期：$(date)"
   echo "目前目錄：$(pwd)"
   echo "========================================="
   echo "磁碟使用量："
   df -h | head -5
   echo "========================================="
   echo "記憶體使用量："
   free -h
   echo "========================================="
4. 按 Esc                         ← 回到一般模式
5. 輸入 :wq 按 Enter              ← 儲存離開
6. chmod +x ~/my_script.sh        ← 加上執行權限
7. ./my_script.sh                 ← 執行
```

如果打錯字需要修改：

```
1. vi ~/my_script.sh              ← 重新開啟
2. 用方向鍵或 j/k 移到要修改的行
3. 按 i 進入插入模式
4. 修改內容
5. 按 Esc → :wq                  ← 儲存離開
```

### 作業第 3 題：chmod 觀念題

第 3 題是觀念題，用 vi 或 nano 建立 txt 檔案都可以：

## 第五部分：nano 與 vi 快捷鍵對照表

| 操作 | nano | vi |
|------|------|-----|
| 開啟檔案 | `nano file.txt` | `vi file.txt` |
| 開始打字 | 直接打 | 按 `i` 進入插入模式 |
| 儲存 | `Ctrl+O` → Enter | `Esc` → `:w` → Enter |
| 離開 | `Ctrl+X` | `Esc` → `:q` → Enter |
| 儲存並離開 | `Ctrl+O` → `Ctrl+X` | `Esc` → `:wq` → Enter |
| 不儲存離開 | `Ctrl+X` → N | `Esc` → `:q!` → Enter |
| 剪下整行 | `Ctrl+K` | `dd` |
| 貼上 | `Ctrl+U` | `p` |
| 複製整行 | 無 | `yy` |
| 復原 | 無 | `u` |
| 搜尋 | `Ctrl+W` | `/關鍵字` |
| 跳到最後一行 | 無 | `G` |
| 顯示行號 | `Ctrl+C` | `:set number` |

## 常見問題

**Q：按了鍵盤但畫面出現奇怪的字母？**
你可能在一般模式下打字了。按 `Esc` 回到一般模式，再按 `i` 進入插入模式。

**Q：不小心改壞了檔案怎麼辦？**
按 `Esc` → 輸入 `:q!` → Enter。這樣會放棄所有修改，回到上次儲存的狀態。

**Q：怎麼知道自己在哪個模式？**
看左下角。顯示 `-- INSERT --` 表示在插入模式，什麼都沒顯示表示在一般模式，顯示 `:` 開頭的文字表示在命令模式。

**Q：vi 和 vim 有什麼差別？**
vim 是 vi 的加強版，功能更多（語法高亮、多層復原等）。大部分 Linux 系統的 vi 指令實際上會啟動 vim。兩者基本操作完全相同。

**Q：作業一定要用 nano 嗎？**
不一定，用 vi 也可以完成所有作業。但作答時請寫出你實際使用的編輯器和對應的快捷鍵。
