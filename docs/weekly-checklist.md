# 每週備課 Checklist

> 每週二上課前確認

## 上課前一天（週一）

- [ ] weekXX/README.md 已更新發布
- [ ] 投影片或教學大綱已準備
- [ ] AWS EC2 環境連線測試正常
- [ ] 上週 PR 已檢閱完畢（或安排助教處理）
- [ ] 本週需要的資料集或腳本已上傳

## 上課當天（週二）

- [ ] 投影片 / 教材最終確認
- [ ] 課堂練習環境確認
- [ ] 點名工具準備
- [ ] 拍照留紀錄（升等佐證）

## 上課後（週二晚或週三）

- [ ] 本週教材 commit + push 到 GitHub
- [ ] 確認學生 fork sync 提醒（如有教材更新）
- [ ] 下週教材開始準備
